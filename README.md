# This was my attempt at building something in C++. 

I have been at C++ for almost 3 years as of writing this README (2nd year of my college if I recall correctly, yes I think semester 3) ,but was mostly used for learning Data Structures and Algorithms.  This was my attempt to build something with it, because for the respect and love I have for this language. It made me from a directionless kid to someone who can write "hello world" and be curious with programming. I was never really a chess prodigy at any point in my life. But the game is kinda fun and addictive, I am writing this project purely for my love for this language, computer science and the people that helped get through my engineering degree.

There can be a lot of untouched scopes in this project, so feel free to fork it, download it, and make it better. I am open to any suggestions.
As a challenge, I asked all my tools(we all know what tools) to act like a Senior software engineer GUIDING A junior dev to build a chess engine. 

As I wrote my previous project, I was subject to constant blank outs on what to write next and heavily relied on these tools. And I think these tools are great help for that,but my dependency on them made me numb to who I am. A kid who loved solving problems. NOT A COPY PASTER BUT A THINKER. 

So I resort to using as little gen-code as possible and use AI as only a tool, when only absolutely necessary. I understand that this might seem a bit contradictory to my earlier statement. But given this age and the value of authenticity of code the current market sees, I have decided to take this step. I still love this language and in no capacity mean to disrespect its community or the developers at all.

I guarantee every developer and community members, that each gen-code logic is fought and understood before adding to this project in any capacity of its impact.
